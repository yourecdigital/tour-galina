# Настройка Telegram-бота для получения заказов

## Шаг 1: Создание бота

1. Откройте Telegram и найдите бота **@BotFather**
2. Отправьте команду `/newbot`
3. Следуйте инструкциям:
   - Введите имя бота (например: "Oktour Orders Bot")
   - Введите username бота (должен заканчиваться на `bot`, например: `oktour_orders_bot`)
4. Скопируйте полученный **токен** (выглядит как: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

## Шаг 2: Получение Chat ID

1. Найдите бота **@userinfobot** в Telegram
2. Отправьте команду `/start`
3. Бот вернет ваш **Chat ID** (число, например: `123456789`)

## Шаг 3: Настройка переменных окружения

1. Создайте файл `.env.local` в корне проекта (если его еще нет)
2. Добавьте следующие строки:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
TELEGRAM_CHAT_ID=ваш_chat_id_от_userinfobot
```

**Пример:**
```env
TELEGRAM_BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
TELEGRAM_CHAT_ID=123456789
```

## Шаг 4: Перезапуск сервера

После добавления переменных окружения перезапустите dev-сервер:

```bash
npm run dev
```

## Проверка работы

1. Откройте сайт: `http://localhost:3000/krasnaya-polyana`
2. Добавьте туры в корзину
3. Перейдите к оформлению заказа
4. Заполните форму и отправьте заказ
5. Проверьте Telegram - вы должны получить сообщение с деталями заказа

## Важно

- Не публикуйте файл `.env.local` в Git (он уже в `.gitignore`)
- Для продакшена добавьте эти переменные в настройки вашего хостинга
- Если переменные не настроены, заказы будут логироваться в консоль сервера
