# Решение проблемы с доступом к сайту

## Проблема: DNS не настроен

Домен `oktour.travel` еще не указывает на IP сервера `185.179.191.27`.

## Временное решение: доступ по IP

### Вариант 1: Прямой доступ по IP (без домена)

Откройте в браузере:
- **http://185.179.191.27** (HTTP)
- **https://185.179.191.27** (HTTPS, если SSL настроен)

### Вариант 2: Настройка hosts файла (для локального использования)

Это позволит использовать домен `oktour.travel` на вашем компьютере.

#### Windows:

1. Откройте Блокнот **от имени администратора**
2. Откройте файл: `C:\Windows\System32\drivers\etc\hosts`
3. Добавьте в конец файла:
   ```
   185.179.191.27 oktour.travel
   185.179.191.27 www.oktour.travel
   ```
4. Сохраните файл
5. Очистите DNS кэш:
   ```powershell
   ipconfig /flushdns
   ```
6. Откройте в браузере: **http://oktour.travel**

#### Linux/Mac:

1. Откройте терминал
2. Отредактируйте файл:
   ```bash
   sudo nano /etc/hosts
   ```
3. Добавьте строки:
   ```
   185.179.191.27 oktour.travel
   185.179.191.27 www.oktour.travel
   ```
4. Сохраните (Ctrl+O, Enter, Ctrl+X)
5. Откройте в браузере: **http://oktour.travel**

**Внимание:** Это работает только на вашем компьютере!

## Постоянное решение: настройка DNS

### Шаг 1: Войдите в панель управления доменом

Зайдите на сайт вашего регистратора домена (где вы покупали домен `oktour.travel`).

### Шаг 2: Найдите настройки DNS

Обычно это называется:
- "DNS Management"
- "Управление DNS"
- "DNS Records"
- "DNS зона"

### Шаг 3: Добавьте A-записи

Добавьте следующие записи:

**Запись 1:**
- **Тип:** A
- **Имя/Хост:** `@` или `oktour.travel` или оставьте пустым
- **Значение/IP:** `185.179.191.27`
- **TTL:** `3600` (или автоматически)

**Запись 2:**
- **Тип:** A
- **Имя/Хост:** `www`
- **Значение/IP:** `185.179.191.27`
- **TTL:** `3600` (или автоматически)

### Шаг 4: Сохраните изменения

После сохранения DNS записи начнут распространяться.

### Шаг 5: Проверка DNS

**В Windows (PowerShell):**
```powershell
nslookup oktour.travel
```

**В Linux/Mac:**
```bash
dig oktour.travel +short
# или
nslookup oktour.travel
```

**Должно вернуть:** `185.179.191.27`

**Время распространения DNS:**
- Обычно: 5-30 минут
- Максимум: до 48 часов

### Шаг 6: После настройки DNS

1. Подождите 5-30 минут
2. Проверьте DNS: `nslookup oktour.travel`
3. Откройте в браузере: **https://oktour.travel**

